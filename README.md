# Basic MVP App (Laravel CMS)

A simple CMS built with Laravel.

## Features
- Dynamic pages (home, about, contact)
- Database-driven content
- Slug-based routing
- MVC architecture

## Tech Stack
- Laravel 12
- MySQL
- Blade Templates
- Git

## Setup Instructions

1. Clone the repo
2. Run composer install
3. Copy .env.example to .env
4. Set DB credentials
5. Run php artisan migrate
6. Run php artisan serve
